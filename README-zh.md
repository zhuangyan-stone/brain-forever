# BrainForever

> 在你和 AI 之间，有一位忠诚的历史记录者，默默了解你的性格、习惯、爱好、职业、做事风格、思维模式、文化水平、艺术品味，甚至还有饮食口味……慢慢建立你的个人特征库，从而让 AI 更好地为你服务。

## 为什么做这个项目

最初想做这个项目，是因为思念我已经走了十多年的父亲。我没办法和他对话了。但我想，终有一天，我也要走。到那时我走了之后，我的孩子，是不是可以在他们想我时，连上服务端，和那个记忆着我半生记忆的智能体说上几句呢？

这就是 BrainForever。

---

BrainForever 是一个会"记住你是谁"的 AI 聊天伴侣。和那些每次对话都从头开始的普通聊天机器人不同，BrainForever 在你和 AI 之间安排了一位低调的"历史记录者"。当你自然地聊天时，这位记录者默默观察并构建你的多维个人特征库——你的沟通风格、价值观、幽默感、专业领域、审美偏好，等等。聊得越多，它就越了解你，AI 的回应也就越贴心、越懂你。

## 为什么选择 BrainForever？

大多数 AI 聊天是**没有记忆的**——每次会话都从零开始，AI 不知道你是谁，也不知道你在乎什么。BrainForever 改变了这一点。

- **它了解的是你，而不仅仅是你的话。** 它能捕捉你的性格特质、决策模式、文化偏好，甚至你的口味。随着时间推移，它会逐步建立起属于你的**个人特征库**。
- **越用越懂你。** 聊得越多，你的个人特征库就越丰富，AI 的回应就越像来自一个真正了解你的人。
- **自然而然地发生。** 不需要填表格、做问卷。只管聊天，历史记录者会处理好一切。
- **你的数据只属于你。** 所有数据都存储在本地——没有云端上传，没有监控，没有第三方画像。

## 功能特性

- **个性化 AI 对话** — AI 通过自然对话逐步了解你独特的性格、沟通风格和偏好，并据此调整回应。
- **流式响应** — 基于 Server-Sent Events (SSE) 的逐 token 实时流式输出，带来流畅自然的聊天体验。
- **可选网页搜索** — 当需要最新信息时，BrainForever 可以搜索网络来补充知识。
- **会话管理** — 按会话组织对话，自动清理闲置会话。
- **深色/浅色主题** — 一键切换主题，白天黑夜都舒适。
- **消息管理** — 按需删除单条消息或整轮对话。
- **优雅关闭** — 收到退出信号后，会妥善保存数据后再退出。

## 环境要求

- Go 1.25.1 或更高版本
- 启用 CGO
- GCC（Windows 上可用 MinGW，Linux/macOS 上可用 gcc）
- SQLite3 开发头文件（详见各平台说明）
- 你需要使用的服务的 API 密钥

## 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/yourusername/BrainForever.git
cd BrainForever
```

### 2. 设置环境变量

| 变量 | 必需 | 说明 |
|---|---|---|
| `DASHSCOPE_API_KEY` | 是（默认） | 文本嵌入 API 密钥 |
| `ZHIPUAI_API_KEY` | 备选 | 备选嵌入服务（设置 `EMBEDDER_PROVIDER=zhipu`） |
| `DEEPSEEK_API_KEY` | 是 | AI 聊天模型 API 密钥 |
| `BOCHA_API_KEY` | 否 | 可选网页搜索的 API 密钥 |
| `PROXY_ADDR` | 否 | 服务端地址（默认 `:8080`） |
| `EMBEDDER_PROVIDER` | 否 | 嵌入服务商：`ali`（默认）或 `zhipu` |

### 3. 各平台准备

**Windows:**
- 安装 [MinGW-w64](https://www.mingw-w64.org/)（例如通过 MSYS2），确保 `gcc` 在 `PATH` 中。
- SQLite3 头文件由 `go-sqlite3` 库自带，无需额外安装。

**Linux (Debian/Ubuntu):**
```bash
sudo apt update
sudo apt install -y gcc libsqlite3-dev
```

**macOS:**
```bash
# Xcode Command Line Tools 包含 gcc 和 SQLite3 头文件
xcode-select --install
```

### 4. 构建并运行

**Windows:**
```batch
b.bat
brain-forever.exe
```

**Linux/macOS:**
```bash
./b.sh
./brain-forever
```

### 4. 打开前端

在浏览器中访问 [http://localhost:8080](http://localhost:8080)。

## API 接口

| 方法 | 路径 | 说明 |
|---|---|---|
| `POST` | `/api/chat` | 发送消息，接收流式 AI 回复 |
| `GET` | `/api/session` | 恢复当前会话的对话历史 |
| `POST` | `/api/history` | 删除一条消息及其关联的 AI 回复 |
| `GET` | `/api/health` | 健康检查 |
