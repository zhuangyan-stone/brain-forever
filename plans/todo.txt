我们现然要新增一个定时慢速任务 ：internal\tasks\excerpt_job.go， 用于在半夜（晚上10到）定时读取用户对话的所有消息，然后生成用户语录摘抄。

我们需要：

一、增加用户语录表 excerpts  和 excerpt_value_dict 表

excerpt_value_dict
- id 整数（不用自增，但可以是主键，SMALLINT）
- value 字符串，限长 38 字符 ，可建唯一值索引 
- value_cn 字符串，限长 40 字符，摘录价值的中文表述
索引： value 

当前固定数据：

excerpts 表 字段：
- id 自增主键
- user_id 外键，引用 users 主键
- chat_id 外键，引用 chat_sessions 主键
- msg_id 摘录自哪条消息，如果来自多条消息，仅记录第一条，不作外键，64位 int
- values  摘录价值对应 ID，无需外键，SMALLINT 的数组，需建 GIN 索引
- content 摘录内容 字符串，限长 380 个字符
- context_summary 所在对话的上下文 字符串，限长 520 个字符
- reason 字符串，限长 400 个字符，描述为会值得摘录
- create_at ，时间类型。默认为记录生成时的当前时间 

索引：user_id, chat_id , msg_id， values、create_at
